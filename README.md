## Getting Started

### Prerequisites

* node.js
* npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lopezrunco/batalla-sarandi-nextjs.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Generate .env file with the public url
   ```sh
   NEXT_PUBLIC_URL = https://example.com.uy
   ```
4. Run development server
    ```sh
    npm run dev
    ```
